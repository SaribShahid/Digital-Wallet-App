## Overview

Digital Wallet is a modern Android application that enables secure and real-time financial transactions. It is designed to simulate a real-world fintech system with features like payments, transfers, and live balance tracking using Firebase.

---

## Key Features

* Secure authentication (Firebase Auth)
* Instant money transfers
* Bill payments & mobile top-ups
* Gift transactions
* QR-based payments
* Real-time transaction history
* Live balance updates

---

## Tech Stack

| Layer    | Technology              |
| -------- | ----------------------- |
| Frontend | Java, XML (Android)     |
| Backend  | Firebase                |
| Database | Cloud Firestore         |
| Auth     | Firebase Authentication |

---

##  Architecture

* Modular Android structure
* Firebase as Backend-as-a-Service (BaaS)
* Real-time data synchronization using Firestore
* Clean separation of UI, logic, and data layers
